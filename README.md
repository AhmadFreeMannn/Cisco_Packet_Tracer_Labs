# Cisco Packet Tracer Labs

This repository contains five beginner-friendly Cisco Packet Tracer labs that cover the fundamental concepts of computer networking.

The labs include:

- LAN Configuration
- Static IPv4 Addressing
- DNS Configuration
- DHCP Configuration
- Static Routing
- RIP Dynamic Routing (RIPv2)
- VLAN Configuration

These labs were created for learning, practicing, and documenting basic networking concepts using Cisco Packet Tracer.
