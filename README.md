# Cisco Packet Tracer Labs

A collection of beginner-friendly Cisco Packet Tracer labs covering essential networking concepts.

## Labs

- Lab 01 – LAN, DNS & Web Server
- Lab 02 – DHCP, DNS & Web Server
- Lab 03 – Multi-Router Static Routing
- Lab 04 – RIP Dynamic Routing
- Lab 05 – VLAN Configuration

## Topics Covered

- IPv4 Addressing
- Subnetting
- DNS
- DHCP
- Static Routing
- RIP
- VLAN
- Network Troubleshooting

## Software

- Cisco Packet Tracer
